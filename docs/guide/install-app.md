# 把 Scholar's Dungeon 主页面安装为应用

本页说明的是：**把 Scholar's Dungeon 主页面安装为应用**，不是把这个 Guide 指南站安装为应用。

主页面地址：<https://scholars-dungeon.karakan.top/>

## 为什么要安装主页面？

安装后，你可以像打开普通应用一样打开 Scholar's Dungeon：

- 桌面端可以从桌面、开始菜单或应用列表进入；
- 手机端可以从主屏幕进入；
- 使用体验更接近独立应用，不需要每次从浏览器收藏夹里找。

## Windows / macOS：Chrome 或 Edge

1. 打开 Scholar's Dungeon 主页面：<https://scholars-dungeon.karakan.top/>。
2. 查看地址栏右侧是否出现“安装”图标。
3. 点击安装图标。
4. 在弹出的窗口中选择“安装”。
5. 安装完成后，可以从桌面、开始菜单或浏览器应用列表打开 Scholar's Dungeon。

如果没有看到安装图标，可以尝试：

1. 点击浏览器右上角的菜单按钮。
2. 找到“应用”“安装此网站为应用”或类似选项。
3. 按提示完成安装。

## Android：Chrome / Edge

1. 用手机浏览器打开 Scholar's Dungeon 主页面。
2. 点击浏览器右上角菜单。
3. 选择“添加到主屏幕”或“安装应用”。
4. 确认名称后完成添加。

## iPhone / iPad：Safari

1. 用 Safari 打开 Scholar's Dungeon 主页面。
2. 点击底部或顶部的“分享”按钮。
3. 选择“添加到主屏幕”。
4. 确认名称后点击“添加”。

## 注意事项

- 请确认你打开的是主页面：`https://scholars-dungeon.karakan.top/`。
- 本 Guide 页面只是说明站，适合阅读教程；真正使用专注功能时请进入主页面。
- 如果浏览器没有出现安装入口，可以先刷新页面，或换用 Chrome / Edge / Safari 尝试。
