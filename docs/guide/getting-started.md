# 开始之前

欢迎来到 ScholarDungeon 的世界！在这里我们把传统的番茄钟体验变成了奇幻的地牢探险。

## 什么是 ScholarDungeon？

ScholarDungeon 是一款将**学习/工作专注**与**放置类地牢探险**结合在一起的游戏化工具。

你可以像使用一般的番茄钟一样设定专注时间，在此期间，探险家角色会在游戏世界中自动进行地牢探索和战斗。当你完成专注目标时，便可获得经验与掉落奖励。

## 适用人群

如果传统的专注应用、打卡工具让你感到枯燥难以坚持，不妨试试把待办事项和学习计划具象化为击杀地牢怪物的过程。

## 准备工作

- 一台支持现代浏览器的设备，或移动端设备。
- 准备好你想在接下来专注的时间内完成的任务。
